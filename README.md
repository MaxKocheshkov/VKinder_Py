# Дипломная работа по курсу Продвинутый Python

## Описание
Используя данные из VK сделан сервис поиска людей, подходящих под условия, 
на основании информации о пользователе из VK:
* возраст,
* пол,
* город,
* семейное положение.

У тех людей, которые подошли по требованиям пользователю, получать топ-3 популярных фотографии с аватара. 
Популярность определяется по количеству лайков и комментариев.

При первом запуске, для доступа к данным ВК, получается безвременный токен, записывающийся в файл `token.ini`.

## Входные данные
id пользователя в ВК, для которого мы ищем пару.
* если информации недостаточно нужно дополнительно спросить её у пользователя.

## Выходные данные
JSON-файл с 10 объектами, где у каждого объекта перечислены топ-3 фотографии и ссылка на аккаунт. Так же полученные 
данные записываются в MongoDB.
